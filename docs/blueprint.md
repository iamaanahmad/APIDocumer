# **App Name**: APIDocumer

## Core Features:

- File Parsing: Automatically parse local `OpenAPI.yaml` files.
- Interactive Documentation Rendering: Render interactive, user-friendly API documentation from the parsed YAML file.
- Endpoint Navigation: Provide a navigable list of API endpoints with clear categorization.
- Detailed Schema Display: Display detailed schemas for each endpoint, including data types and constraints.
- Example Request/Response Generation: Generate example request and response payloads based on the schema.
